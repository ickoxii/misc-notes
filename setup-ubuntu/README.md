## Setup for C++ Development

```bash
# Update our package manager if needed
apt update

# Grab c++ related packages
# g++, gcc, gdb, make
apt install build-essentials

# Get a text editor
apt install vim

# Memory Checker
apt install valgrind

# Git integration
apt install git-all

# Terminal multiplexer
apt install tmux
```
